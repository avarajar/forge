A. Input-Based (recommended)

Ask "what do you have?" — Figma link, description, sketch, screenshot. Forge detects your stack (Tailwind? shadcn? Figma MCP connected?) and chains the right tools. One screen, adapts to input.

🎨 Design Task
CHOOSE A WORKFLOW
Figma → Code
Paste a Figma link, get React/Vue/Svelte components
Description → Prototype
Describe what you want, AI generates a working prototype
Style Guide Setup
Define or import colors, typography, spacing, components
Visual QA
Compare implementation vs design, catch visual regressions

B. Workflow Picker

Choose what you want to do: Figma→Code, Description→Prototype, Style Guide Setup, Visual QA. Each is a predefined workflow. Clearer but more rigid.

C. Universal Input
